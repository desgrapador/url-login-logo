# URL Login Logo Plugin

## Description
URL Login Logo is a WordPress plugin that allows you to customize the logo on the login page and change the link to the homepage.

## Installation
1. Download the plugin zip file from the [URL Login Logo Plugin](https://martinenrique.com/url-login-logo/) page.
2. Log in to your WordPress admin panel.
3. Go to **Plugins** > **Add New**.
4. Click on the **Upload Plugin** button and select the zip file you downloaded.
5. Click **Install Now** and then **Activate Plugin**.

## Configuration
1. After activating the plugin, a new field will appear under **Settings** > **General**.
2. Scroll down to the **URL Login Logo** section.
3. Enter the URL of the logo image you want to use to replace the default login logo.
4. Set the width and height of the custom logo in pixels.
5. Click **Save Changes**.

## Usage
Once the plugin is configured, the custom logo will be displayed on the login page according to the specified dimensions. The login logo will also be linked to the homepage of your WordPress site.

Note: If you leave any of the width or height fields empty, the logo will maintain its original aspect ratio.
